# 🚀 Zuno – Retail Operations Platform

Zuno is a full-stack retail platform with a **customer-facing storefront** and a **business operations dashboard**, built to simplify real-world shop management.

It enables small businesses to handle **sales, billing, inventory, credit, orders, and finance analytics** in one unified system, with real-time workflows powered by Firebase.

## 🧩 Features

### 🛍 Customer-Facing Storefront
- Product browsing with dynamic pricing
- Cart and order placement
- Real-time order tracking
- Delivery / Pickup workflow

### 🏪 Business Operations Dashboard
- Sales and billing management
- Inventory tracking with unit normalization (kg/gram/piece)
- Credit management (customer-wise tracking)
- Order status management
- Payment mode tracking (cash, online, etc.)

### 📊 Finance & Analytics
- Revenue, profit, and expense tracking
- Cash flow visibility
- Outstanding credit monitoring
- Transaction-level insights

### 🤖 AI & Smart Workflows
- AI-assisted sales summaries
- Structured voice/text order extraction
- Faster order entry for real shop use cases

### ⚙️ System Design
- Multi-user architecture
- Firebase Authentication for secure access
- Firestore user-scoped data models
- Public shop access with controlled backend logic
